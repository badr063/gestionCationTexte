Documentation - Gestion de Cahier de Texte

Une application pour faciliter la gestion pédagogique.

Introduction

L'application Gestion de Cahier de Texte est conçue pour optimiser la planification, le suivi et la gestion des activités pédagogiques. Elle permet aux enseignants de saisir leurs cours, aux chefs de département de superviser les programmes, et aux responsables administratifs de centraliser les informations.

Fonctionnalités principales

  • Gestion des cours : Ajouter, modifier ou supprimer des informations sur les cours.
  • Génération de fiches pédagogiques : Créer automatiquement des fiches pédagogiques en PDF.
  • Saisie des données pédagogiques : Entrer des informations comme les objectifs et ressources.
  • Gestion des relations : Relier enseignants, responsables de classe et chefs de département.

Architecture technique

Technologies utilisées :

  • Langage : Java
  • Bibliothèque : iText (génération de PDF)
  • IDE : Eclipse
  • Le fichier jar

Structure logicielle :

- La classe FichePedagogique centralise les données des cours.
- La classe ChefDepartementUI fournit une interface graphique intuitive.

Mode d'emploi

  1. Lancer l'application.
  2. Inscrire si vous n'avez pas de compte.
  3. saisissez votre mail et votre mot de passe.
  4. Si c'est le chef departement:
  5. Cliquer sur "Générer une Fiche Pédagogique".
  6. Remplir les champs requis (date, contenu, ID).
  7. Cliquer sur "Générer PDF" pour créer une fiche pédagogique.

Exemple pratique

Un enseignant entre le contenu d'un cours, le chef de département valide les informations, et une fiche pédagogique est générée et partagée avec les étudiants.
